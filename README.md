# Amazon Sales Dashboard (Excel Project)

## Project Overview
This project analyzes Amazon sales transaction data and transforms raw data into an interactive dashboard using Microsoft Excel.

The dashboard helps business teams monitor sales, delivery efficiency, cancellations, and regional performance.

---

## Objective
- Clean raw sales data
- Perform business analysis using Pivot Tables
- Apply XLOOKUP for cross-sheet data retrieval
- Create KPI cards for decision making
- Design an interactive dashboard

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP
- Pivot Charts
- Conditional Formatting
- Slicers

---

## Dataset Features
The dataset contains:
- Order Date
- Delivery Date
- Cancel Date
- Region
- Product Category
- Payment Method
- Sales Amount
- Delivery Status

---

## Key Analysis Performed

### Sales Analysis
- Total Sales Amount by Region
- Product-wise Revenue Analysis
- Monthly Sales Trend

### Delivery Analysis
- Average Delivery Time
- Fast vs Slow Deliveries
- Delivery Status Distribution

### Business KPIs
- Total Revenue
- Fast Delivery Percentage
- Top 3 Performing Regions
- Highest Selling Product

---

## KPI Results
- Total Revenue: 13656
- Fast Deliveries: 33.28%
- Highest-selling Product: Electronics
- Best Region: North

---

## Key Insights
1. Electronics generated highest revenue.
2. North region performed best in sales.
3. Cancellation rate was approximately 30%.
4. Delivery performance needs improvement in East and Central regions.

---

## Recommendations
- Improve logistics for slow-delivery regions
- Reduce cancellations through better inventory planning
- Focus on high-performing product categories
